About Account

# switch to root
sudo du
# switch to other user
su username
# modify password
passwd username

crontab

# edit tasks
crontab -e

Command format:

字段 * * * * * Command
含义 分钟 小时 天 月 星期 任务命令
范围 0~59 0~23 1~31 1~12 0～7（0和7表示星期天）

Characters meaning:

  • * all values
  • , and
  • - from to
  • / interval
