Quick Start

Create a new post

hexo new "My New Post"

More info: Writing

Run server

hexo server

More info: Server

Generate static files

hexo generate
hexo g

More info: Generating

Deploy to remote sites

hexo deploy
hexo d

More info: Deployment

Or you can deploy it one step

npx hexo clean && npx hexo g -d
